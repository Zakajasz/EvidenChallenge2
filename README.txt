# Walidator Harmonogramu Czasu Pracy

## Opis projektu

Prosty walidator harmonogramu czasu pracy napisany w języku Java. Program czyta harmonogram czasu pracy z pliku `harmonogram.txt` i wykonuje różne walidacje dotyczące ilości godzin pracy, przerw między dniami, nadgodzin itp.

## Jak uruchomić program

1. Sklonuj to repozytorium na swój komputer:
git clone https://github.com/Zakajasz/EvidenChallenge2

2. Przejdź do folderu z projektem:
cd EvidenChallenge2

3. Skompiluj kod źródłowy:
javac Main.java

4. Uruchom program:
java Main

5. Wykonaj polecenia z konsoli zgodnie z wyświetlonymi instrukcjami.

## Struktura pliku `harmonogram.txt`

Plik `harmonogram.txt` powinien zawierać dane dotyczące harmonogramu czasu pracy. Każda linia w pliku reprezentuje jeden dzień i powinna mieć format:
numer_dnia, dzień_tygodnia, liczba_godzin
Na przykład:
1, poniedziałek, 8
2, wtorek, 7
...
